# Schnider Sedna Wallswitch ZigBee board
To make regular switch ZigBee-poewered and keep it compact

## Hardware
- E18-MS1-PCB(CC2530 board)
- button
- led
- capacitor
- CR2032 battery holder(multicomp bc2001)
- battery

### Additions
