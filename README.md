# Schneider Sedna Wallswitch ZigBee board
To make regular switch ZigBee-powered and keep it compact

![](assets/render.png)

## Hardware
- E18-MS1-PCB(CC2530 board)
- button
- led
- capacitor
- CR2032 battery holder(multicomp bc2001)
- battery
  
![](assets/sch.png)  

### Additions
Pin length - 18mm
